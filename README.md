# wechat_auto
通过wxpy拉取微信好友列表保存至本地Excel 微信自动发送消息至好友，消息内容包括： 天气、http://wufazhuce.com 每日一言

开发环境：Python3.7.2 默认安装pip
开发工具VSCODE

需要安装组件：

#读取excel文件

pip install xlrd

#写excel文件

pip install xlwt

#定时任务框架

pip install apscheduler

#HTML或XML文件中提取数据的Python库

pip install BeautifulSoup

#微信API封装库

pip install wxpy
